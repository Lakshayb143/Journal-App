Welcome to My Journaling App API

Thank you for visiting! This API is designed to power a journaling application where users can manage their journal entries efficiently.

Key API Features:

  • Authentication: Secure user login and registration using MongoDB.
  • Journal Entries: Create, update, delete, and retrieve journal entries.
  • User Management: Manage user details and associated journal entries.
